# Thesis Presentation Slides

A Reveal.js slide deck for thesis/supervisor meetings presenting the domain-invariant speech deepfake detection research.

## Requirements

⚠️ **Internet connection required** — This presentation loads assets from external CDNs:
- [Reveal.js](https://cdn.jsdelivr.net/npm/reveal.js@4.5.0/) (presentation framework)
- [Google Fonts](https://fonts.googleapis.com/) (Source Serif 4, Source Sans 3, JetBrains Mono)

The slides will render unstyled/broken in offline or network-restricted environments.

## Usage

Open `presentation.html` in a web browser.

### Navigation
- **Arrow keys** or **Space**: Navigate slides
- **Esc**: Overview mode
- **F**: Fullscreen

## Note

The final slide contains hardcoded references to this repository and the associated W&B project. These are intentional for the thesis project context.
